# HTML Briefing

本專案用來製作一份 HTML 簡報。目標是把簡報內容、素材與最終輸出分開管理，方便後續逐步撰寫、設計、預覽與交付。

## 目的

- 建立可用於瀏覽器播放或展示的 HTML 簡報。
- 保留原始資料、整理後素材與最終輸出版本。
- 讓後續工作能清楚追蹤目前進度與待辦事項。

## 結構

- `materials/`: 原始資料、參考文件、使用者提供的內容。
- `assets/`: 可重用的圖片、字型、樣式、腳本或整理後素材。
- `outputs/`: 最終 HTML 簡報與可交付版本。
- `README.md`: 專案概覽。
- `AGENTS.md`: AI agent 工作規則。
- `TODO.md`: 待辦事項與下一步。
- `PROGRESS.md`: 進度紀錄與決策。

- 下一步: 提供簡報主題、內容素材或大綱，開始規劃 HTML 簡報結構。
